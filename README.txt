ABOUT
─────
This is a beautiful, interactive digital birthday card created for Gyanvi!
It features premium animations, confetti effects, background music, and a warm 
birthday message with personalized wishes.

FEATURES
────────
✨ Premium Gradient Design
   - Beautiful purple and pink gradient background
   - Smooth animations and transitions
   - Modern card layout with rounded corners

🎉 Interactive Elements
   - Confetti explosion effect
   - Birthday music player
   - Keyboard shortcuts (Space for confetti, M for music)
   - Double-click to trigger confetti

🎈 Animations
   - Bouncing title text
   - Pulsing name
   - Spinning profile image
   - Floating balloons
   - Wiggling cake decoration
   - Sliding wish items

🎵 Music Integration
   - Background birthday music
   - Play/Pause functionality
   - Auto-play with confetti

📱 Responsive Design
   - Works on desktop, tablet, and mobile
   - Optimized layouts for all screen sizes

FILE STRUCTURE
──────────────
Birthday-Gyanvi/
├── index.html           - Main invitation page
├── style.css            - Premium design and animations
├── script.js            - Interactive functionality
├── assets/
│   ├── gyanvi.jpg       - Birthday person's photo (replace with actual image)
│   ├── balloons.png     - Balloon decoration
│   ├── cake.png         - Cake icon
│   ├── confetti.js      - Confetti particle system
│   └── birthday-music.mp3 - Background music (add your own)
└── README.txt           - This file

HOW TO USE
──────────
1. Replace the placeholder images in the assets/ folder:
   - gyanvi.jpg: Upload a photo of the birthday person (200x200px recommended)
   - cake.png: Cake decoration image (80x80px recommended)
   - balloons.png: Balloon decoration (optional)

2. Add your birthday music:
   - Replace birthday-music.mp3 with an actual birthday song
   - Supported formats: MP3, WAV, OGG

3. Customize the message:
   - Edit the greeting message in index.html
   - Add personalized wishes to the wishes section

4. Open index.html in a web browser to see the card!

KEYBOARD SHORTCUTS
──────────────────
Spacebar  → Trigger confetti explosion
M         → Play/Pause birthday music
Double-Click → Trigger confetti (on card area)

BUTTONS
───────
🎵 Play Music    → Toggle background music on/off
🎉 Celebrate!    → Trigger confetti explosion and auto-play music

CUSTOMIZATION
─────────────
To personalize this card:

1. Change the name:
   - Edit "Gyanvi" in the <h1> and <p> tags in index.html

2. Update the message:
   - Edit the message text in the message-section div

3. Modify wishes:
   - Update the emoji and text in each wish-item div

4. Change colors:
   - Update gradient colors in style.css
   - Modify the color arrays in script.js for confetti

5. Adjust animations:
   - Modify @keyframes in style.css
   - Adjust animation duration and timing functions

TROUBLESHOOTING
───────────────
If images don't appear:
- Make sure image files are in the assets/ folder
- Check that filenames match exactly (case-sensitive)
- Use common formats: JPG, PNG, GIF

If music doesn't play:
- Ensure birthday-music.mp3 exists in assets/ folder
- Some browsers require user interaction before playing audio
- Check browser console for any error messages

If animations lag:
- The animations are GPU-accelerated
- Update to the latest browser version
- Close unnecessary tabs/applications
